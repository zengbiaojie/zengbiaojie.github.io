教程

环境准备

Hugo

官网：https://gohugo.io/

Github: gohugoio/Hugo

进入Releases下载最新版即可

博客搭建

创建博客项目

解压下载得到的 Hugo 压缩包，并得到其中的 hugo.exe 文件

创建一个新的文件夹作为个人博客文件夹

将 hugo.exe 放入创建的文件夹中，并打开cmd窗口，运行

hugo new site xxx(你的博客项目名称)

上述命令会自动创建博客项目文件夹

将 hugo.exe 复制到博客项目文件夹中方便后续使用

进入项目路径 

cd xxx

选择一个主题

主题网站：themes.gohugo.io

选择主题后，根据每个主题的教程进行配置

这里以 reimu 主题为例

下载并使用主题 

git init
git submodule add https://github.com/D-Sketon/hugo-theme-reimu.git themes/reimu
echo 'theme = "reimu"' >> hugo.toml

先初始化git仓库后使用 git submodule 添加主题内容，并将所使用的主题写在 hugo.toml 配置中

运行博客 

hugo server

运行上述命令启动博客服务，就可以在图中红框位置找到博客运行地址，使用浏览器访问该地址即可查看博客内容
